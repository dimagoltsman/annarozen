## How to Use

1. **Open the site**: Just open `index.html` in your browser
2. **Customize content**: Edit the HTML to update text, links, etc.
3. **Add real images**: Replace placeholder images with actual photos
4. **Update links**: Add real social media and booking URLs

## Customization Guide

### Replace Avatar Image
In `index.html`, line with id `avatar-img`:
```html
<img src="YOUR_AVATAR_IMAGE.jpg" alt="Anna Rozen Golzman" id="avatar-img">
```

### Update Social Links
Find the contact links section and update href attributes:
```html
<a href="https://instagram.com/YOUR_USERNAME" class="link-button instagram">
<a href="https://facebook.com/YOUR_PAGE" class="link-button facebook">
<a href="YOUR_BOOKING_URL" class="link-button booking">
<a href="https://wa.me/YOUR_PHONE_NUMBER" class="link-button whatsapp">
```

### Add Gallery Images
Replace placeholder images in each gallery section:
```html
<img src="path/to/your/image.jpg" alt="Description">
```
